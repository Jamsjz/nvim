2. Clone this repo in $HOME/.config folder
3. Remove all local cache and share data of nvim.

* Keymaps
All your custom keymaps for plugins or anything here
== Keymaps listings ==
1. Buffers:
   Tab for buffer next and shift tab for previous
   Also C - w changes to C - backspace
2. Cursor:
   When typing if you want your cursor to nudge just a bit maybe out of a parenthesis
   use control jklh to move cursor without leaving insert mode
3. Code Runner:
   Just some keybindings for the plugin code_runner.nvim
4. Warp movement:
   When typing your editor can run out of column_width, to deal with it editor use wrap.
   But they treat it as a line so one line is wrapped in two but when pressing hjkl it goes to the actual next line
   But you might want to go one step above in the wrap.

* Options
Options are automatically loaded before lazy.nvim startup
Default options that are always set from [[https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/options.lua][LazyVim Options]]
== Options Listing ==
1. UI:
   - Enable color
   - Max Item to show in a pop up menu
   - Max Item to show in a command menu
2. Backspace and Word Realilzing:
   It detects words with hyphon and when doing ^ BS  it doesn't remove the whole word
   but only till hyphon.
3. Cursor Unchanged after quiting
4. Markdown wrap and linebreak
5. Neovide transparency
